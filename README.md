# translator

- Clone
- Run `python3 -m venv venv`.
- Run `. venv/bin/activate`.
- Run `npm i`
- Run `python install -e .`.
- Run `run` to execute script.

- Run `animals 'savethis.txt' 'savenospaces.txt'` to scrape animals.
